# Game screenshots

Drop high-resolution gameplay screenshots here.

To migrate from the existing press kit:

```
cp dcsale/coblade-game-press/images/screenshots/01.png .
cp dcsale/coblade-game-press/images/screenshots/02.png .
cp dcsale/coblade-game-press/images/screenshots/03.png .
cp dcsale/coblade-game-press/images/screenshots/04.png .
```

Recommended:
- 1920×1080 or higher
- PNG format (preferred over JPG for game UI clarity)
- Numeric naming: `01.png`, `02.png`, ... — keeps gallery order predictable
- 5-10 strong screenshots across the game's lifetime; mix wide-vista, feature highlight, and in-action shots
